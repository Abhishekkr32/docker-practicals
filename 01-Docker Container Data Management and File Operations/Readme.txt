Docker Practical – Container File Operations
Objective

To understand how Docker containers work and how to manage files inside a container.

Description

In this practical, we:

Created and started a Docker container using Ubuntu.
Passed an environment variable inside the container.
Verified the environment variable.
Created a new folder inside the container.
Created a text file inside that folder.
Stopped and restarted the container.
Copied a file from the container to the host system (Desktop).
Copied a file from the host system back into the container.
Verified the file inside the container.

Concepts Learned

Docker container lifecycle (run, start, stop)
Environment variables in Docker
File creation inside container
Data transfer between host and container
Container data persistence

Conclusion

This practical helped in understanding basic Docker container operations and how files can be managed and transferred between the container and the host system.
