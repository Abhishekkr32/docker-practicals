Docker Practical – Volume Management

Objective---
To understand Docker volumes and how they provide persistent storage.

Description--
In this practical, we created a Docker volume and attached it to a container. We verified that the volume was properly mounted inside the container. After removing the container, we manually removed the volume and cleaned up unused volumes.

Concepts Learned---
Docker volume creation
Volume listing and inspection
Mounting volume to container
Container inspection
Removing container and volumes
Data persistence using volumes

Conclusion---
Docker volumes allow data to persist independently of containers. Even if a container is deleted, the volume remains until it is manually removed.
